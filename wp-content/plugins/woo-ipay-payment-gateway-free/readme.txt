It's a WooCommerce extension allowing clients make payment with iPay.

== Description ==
WooCommerce payment gateway plugin for iPay/eLipa which accept payments from clients in Kenya, Uganda, Tanzania and Togo. Fully supported with WordPress and WooCommerce latest versions. iPay latest API 3.5 compatible. These are the supported channels on each of those countries:
* Kenya: M-Pesa, Airtel Money, Equitel, VISA, Mastercard, pesalink, kenswitch and mVisa
* Tanzania: Vodacom, VISA and Mastercard
* Uganda: MTN, VISA and Mastercard.
* Togo: Flooz, T-Money, VISA and Mastercard.

= Features =
* Setup the plugin depending on country of operation.
* Enable/Disable Test Mode
* Checkout mode: iFrame or Redirect

= How to Setup Account with iPay = 
[Step by step iPay Account Setup Guide](https://ipayafrica.com/for-business/)

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After activation, you can set options from `WooCommerce -> Settings -> iPay` menu
